# Legal Olympiad – Frontend Assignment

This project is a modern and responsive homepage for the Legal Olympiad platform. It is designed to present sections like blogs, webinars, rankings, and updates in a clean and user-friendly way.

---

## 🚀 Live Demo

https://olympiads-tawny.vercel.app/

## 📂 GitHub Repository

https://github.com/lokeshhans/olympiads

---

## 🧠 Approach

I focused on building a clean and simple UI with a proper layout structure. The page starts with a hero section and then shows key areas like rankings, blogs, and webinars.

The goal was to make it feel like a real platform that law students can use easily.

---

## ✨ Features

* Responsive design (mobile + desktop)
* Clean and modern UI
* Reusable components
* Multiple sections:

  * Blogs / Articles
  * Webinars & Seminars
  * News & Updates
  * Student Highlights
  * College Rankings
* Navigation using React Router
* Basic login/logout (using localStorage)
